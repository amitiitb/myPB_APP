# Typography Guide

## Font Size Standards

### Headings
- **H1 (Page Title)**: 20px, fontWeight: '700' (bold)
- **H2 (Section Title)**: 18px, fontWeight: '700' (bold)
- **H3 (Card Title)**: 15px, fontWeight: '600' (semibold)

### Body Text
- **Body Large**: 15px, fontWeight: '600' (semibold)
- **Body Regular**: 13px, fontWeight: '400' (regular)
- **Body Small**: 12px, fontWeight: '500' (medium)

### UI Elements
- **Button Text**: 15px, fontWeight: '600' (semibold)
- **Label Text**: 13px, fontWeight: '500' (medium)
- **Hint/Placeholder**: 13px, fontWeight: '400' (regular)
- **Caption**: 12px, fontWeight: '500' (medium)
- **Section Header (caps)**: 12px, fontWeight: '600' (semibold), textTransform: 'uppercase'

## Font Weights
- **Regular (400)**: For body text and descriptions
- **Medium (500)**: For labels and small text
- **Semibold (600)**: For interactive elements and emphasis
- **Bold (700)**: For headings and titles

## Color Standards
- **Primary Text**: #111827 (Dark gray/black)
- **Secondary Text**: #6B7280 (Medium gray)
- **Tertiary Text**: #9CA3AF (Light gray)
- **Muted Text**: #D1D5DB (Very light gray)
- **Accent**: #7C3AED (Purple - primary brand color)
- **Error**: #EF4444 (Red)

## Applied Consistency
✅ Settings Page - Updated to match app standards
✅ Dashboard Screen - Uses consistent sizing
✅ Orders Page - Uses consistent sizing
✅ Other Tab Screens - Use consistent sizing

## Implementation Notes
- All fontSize values are in pixels
- Maintain proper line-height with marginTop/marginBottom adjustments
- Use consistent spacing (12px, 14px, 16px, 18px increments)
- Keep color palette limited to defined standards
